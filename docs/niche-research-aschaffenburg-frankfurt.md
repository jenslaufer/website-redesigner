# Website Redesign Cold Outreach: Niche Analysis

## Top 3 — Start Monday

### 1. Steuerberater (9/10)
- ~1,600 practices in region (9,000+ in Hessen)
- 50-65% outdated websites
- Budget: €800-2,000
- Decision-maker = owner, direct email via Steuerberaterverzeichnis
- **Action:** Pull 50 from official registry within 30km. Run pipeline. Send 10-15 emails/day.
- **Expected:** 5-12% response rate

### 2. Rechtsanwälte — Solo/Small Only (9/10)
- 3,000-5,000 solo/small practices in Frankfurt region
- 55-70% outdated (notorious for static sites)
- Budget: €1,200-2,000 (highest per-deal)
- Filter via anwalt.de: solo practitioners, family/tenancy/injury law
- **Action:** Pull 40-50 from anwalt.de, avoid CBD large firms. Send 10 emails.
- **Expected:** 4-8% response, but highest deal value

### 3. KFZ-Werkstaetten (7.5/10)
- 600-900 independent workshops (78 in Aschaffenburg alone)
- 65-75% outdated, overlooked by agencies
- Budget: €700-1,500
- **Phone-first approach** — owner answers directly, follow up with screenshot
- **Action:** Pull 50-60 from 11880-werkstatt.com, filter out franchises. Call first, email screenshot.
- **Expected:** 8-15% on calls, 3-6% on email

## Avoid Initially

- **Friseursalons:** Budget below minimum (€300-600), template saturation
- **Restaurants:** Thin margins, Wix/Lieferando replaced websites
- **Aerzte/Zahnaerzte:** High competition from medical-specific agencies
